# Frontend

这是一个基于 Vue 3 + Vite 的前端示例工程，用于展示医院预约系统首页并调用后端接口。

## 目录说明

- `src/`: 前端源码
- `package.json`: 前端依赖和脚本

## 运行环境

- Node.js 18+
- npm 9+

## 安装依赖

```bash
npm install
```

## 启动开发环境

```bash
npm run dev
```

启动后默认地址为 `http://localhost:5173`。

## 页面能力

- 展示后端健康状态
- 获取医生列表并渲染到首页

请确保后端已先在 `http://localhost:8080` 启动，否则页面会提示接口请求失败。
