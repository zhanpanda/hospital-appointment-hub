# Frontend

这是一个基于 Vue 3 + Vite 的前端联调工程，当前已经直接接入医院预约系统后端的认证、科室、医生和就诊人接口。

## 目录说明

- `src/`: 前端源码
- `package.json`: 前端依赖和脚本

## 运行环境

- Node.js 18+
- npm 9+

## 安装依赖

```bash
npm install
```

## 启动开发环境

```bash
npm run dev
```

启动后默认地址为 `http://localhost:5173`。

## 页面能力

- 手机号 / 邮箱注册与登录
- 自动持久化 token，并为业务接口附带 `Bearer` 鉴权头
- 拉取科室列表
- 按科室分页查询医生并查看详情
- 新增、编辑、删除、设默认就诊人

## 联调说明

- Vite 已将 `/api` 代理到 `http://localhost:8080`
- 启动前请确保后端服务与数据库已准备完成
- 预约和排班接口后端控制器暂未开放具体能力，因此当前前端尚未接入这两块
