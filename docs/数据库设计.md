| `uk_doctor_date_period` | schedules      | 同一医生同一天同时段只能有一条排班   |
| ----------------------- | -------------- | ------------------------------------ |
| `uk_member_dept_date`   | appointments   | 同一就诊人同一天同一科室只能预约一次 |

    │                           │
    └── 1:N → appointments ────┘
                    │
departments (科室)  │
    │               │
    └── 1:N → doctors (医生)
                  │
                  ├── 1:N → schedules (排班)
                  │              │
                  └── 1:N → appointments
